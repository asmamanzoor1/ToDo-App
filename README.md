# 📝 Java To-Do List App

This is a simple **To-Do List application** built in **Java** as part of my **CISC 3130: Data Structures** class. 

---

## 💻 Features
- 📋 Add new tasks to your list
- ✅ Check off tasks when done
- 🗃️ Stores tasks using Java's `ArrayList`
- 🧠 Practice with object-oriented programming concepts

---

## 🛠️ Tech Stack
- **Language:** Java  
- **Concepts:** ArrayLists, Classes, Loops, Conditionals  
- **IDE Used:** Netbeans
